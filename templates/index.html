How Does It Work

Let us now understand the functionality, process, or mechanism behind our system.

01. Upload Video File

Our user-friendly interface allows users to upload video file from which they want to track the object.

02. Select the Object

Allows users to select a particular object that they want to track within the video file, either by selecting from a list of common objects or by manually tagging the object in the initial frame.

03. Auto Generation

Our system will detect and identify the selected object and then track it as it moves across differnt frame. Then cuts and saves video clips based on object's appearance in the footage from its starting point to its last detected frame. It will also merge all the segmented clips into one single video providing user a continuous view of all the instances of the object.

04. Results

Our System will store the output generated in a database that allows users to save and retrieve videos and then allows users to view and export the generated video of the continuous instances of the object.
